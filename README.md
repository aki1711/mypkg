## このリポジトリについて
このリポジトリはロボットシステム学の講義で使用したROS2のパッケージです。
talkerとlistenerの2つのノード、talk_listen.launch.pyというlaunchファイルが使用できます。

## インストール
* 以下のコマンドを実行して、リポジトリをクローンします。
```
$ git clone https://github.com/aki1711/mypkg.git
```

* そのあと、クローンしたリポジトリに移動します。
```
$ cd mypkg
```
